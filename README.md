# persistent-try-bugs

Personal repository used with nix for fixing bugs related to persistent

## Sqlite refresher

sqlite3 db_name

Some commands:

* .tables
* .schema <table_name>
* .quit

## Postgresql refresher (psql)

sudo su
su postgres
psql -U postgres

Some commands:

* \l   (List of databases)
* \d   (List of tables)
* \c db_name (Connect to database named db_name)
